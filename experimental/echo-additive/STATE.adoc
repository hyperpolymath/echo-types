== Phase R1 — Variance gate (STOP)

=== The composition candidate

`gduplicate keep g : GEcho keep → GEcho g` instantiates, since `GEcho keep = A`, to:

----
A → D_g A
----

*Direction:* this is a UNIT in the MONADIC direction (`Id ⇒ D`).

*What a graded comonad requires:*

* Counit ε : D₀ A → A (extraction — goes D → Id)
* Comultiplication δ : D_{r+s} A → D_r(D_s A) (grade splitting — goes D_{r+s} → D_r ∘ D_s)

*The obstruction:* `gduplicate keep g` gives η (A → D_g A), not δ. Any "gradewise
section" that would supply D_r ∘ D_s ≅ D_{r+s} in the additive Grade setting uses
the combining direction of Echo-comp-iso — which is the monadic multiplication μ,
not the comonadic comultiplication δ. Using it as δ conflates comonad with monad and
breaks coeffect tracking (resources measured in the wrong direction).

This is exactly the obstruction R-2026-05-18 diagnosed as an *absence* ("no nested
D_r D_s"). The R1 gate names the *cause*: the available combining map has monadic
variance, so the absence cannot be filled without inverting the direction.

==== What this is and why

The variance analysis established that the combining direction of the composition
map (Echo-comp-iso "from") is monadic and the splitting direction is comonadic.
Rather than choosing one and discarding the other before construction, both are
pursued so that the comparison can determine what each reading captures that the
other cannot — specifically whether *recoverability* (the section/retraction
structure, the map that lets you ask "given the lossy output, what inputs were
possible") is naturally monadic, naturally comonadic, or genuinely needs both.
That question cannot be answered by reasoning alone; it requires the laws to be
attempted in Agda.

==== Primary line: graded monad

*Structure:* laxator `φ_{r,s}(A) : D_r(D_s A) → D_{r+s} A`, natural in A,
constructed via Echo-comp-iso's combining ("from") direction.

*Variance:* monadic (μ direction). Accepted explicitly. This reading gives
algebraic composition D_r ∘ D_s and is where D_r ∘ D_s ≅ D_{r+s} lives.

*First target (R3):* discharge the three coherence laws in `--safe --without-K` Agda:

. Pentagon (associativity):
  `φ_{r,s+t} ∘ D_r(φ_{s,t}) = φ_{r+s,t} ∘ φ_{r,s}(D_t(-))`.
. Left unit:
  `D_{fin 0}(D_r A) → D_r A` reduces to the `D_{fin 0} ≅ Id` coercion.
. Right unit:
  `D_r(D_{fin 0} A) → D_r A` reduces to `D_r` applied to the same coercion.

=== R2 — `GradedCarrier.agda` (the ℕ∪{∞} carrier)

`D : Grade -> Set -> Set`, `D (fin 0) A = A` (= Id), `D inf A = Top` (= collapse),
`D (fin (suc n)) A = Top × D-fin n A` — the **payload-independent ⊤-padded tower**.
The R2 design question (what is a finite layer?) is answered by an *obstruction
that is itself the finding*: instantiating `Echo-comp-iso` needs a codomain map
`g` and base point `y` that the bare signature `Grade -> Set -> Set` does not
supply; the only generic map is `id/id`, whose residue is doubly-contractible
(refl-only). So the only K-free finite layer is the `⊤` factor, hence
**`D (fin n) A ≅ A` for every finite n** — the grade counts layers, the layers
carry no recoverable content. This information-inertness IS the carrier-side
shadow of the R1 variance finding. The content-bearing alternative
`Σ A (λ _ → …)` was rejected (it gives `A^((m+1)(n+1))`, which does not fuse with
addition's `A^(n+m+1)`). Keystone: `D-fin-+-split m n : D-fin (m+n) A ≡ D-fin m
(D-fin n A)` — the *definitional* additive shadow of `Echo-comp-iso`; φ reads its
`sym` (combine), δ reads it forward (split).
