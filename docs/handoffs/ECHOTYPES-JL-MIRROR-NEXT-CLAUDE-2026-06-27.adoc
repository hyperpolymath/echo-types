THE SOURCE OF TRUTH (echo-types, already merged to main)

  `Maybe ‚Ñï` (`nothing` = ‚àû), plus a worked `countAggregator`.

  distinct values with the same aggregate) ‚áí no section, via `no-section-of-collapsing-map`.

   no-canonical-disaggregation (exhibit that no left inverse / section exists on a concrete
   collision ‚Äî a chosen "raise" cannot satisfy `raise ‚àò pairsum == id` because two pairs map
   to the same total).
