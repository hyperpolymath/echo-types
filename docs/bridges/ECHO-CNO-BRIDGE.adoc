# Echo Types to CNOs Bridge

## Overview

This document establishes the theoretical bridge between **Echo Types** (from the echo-types repository) and **Certified Null Operations (CNOs)** (from the absolute-zero repository).

## Key Insight

**CNOs are a special case of echo types where the computation is the identity function.**

- **Echo Types**: `Echo f y := Σ (x : A) , (f x ≡ y)` - fibers over arbitrary functions
- **CNOs**: Programs that preserve state completely (`f = id`)
- **Bridge**: `Echo id s` represents all states that map to `s` under identity (i.e., only `s` itself)

## Mathematical Foundation

### Echo Type Definition

```agda
Echo : ∀ {a b} {A : Set a} {B : Set b} (f : A → B) → B → Set (a ⊔ b)
Echo {A = A} f y = Σ A (λ x → f x ≡ y)
```

### CNO as Echo Type

For a CNO, the computation function `f` is the identity:

```agda
cno-identity : ProgramState → ProgramState
cno-identity = id

-- CNO echo type
Echo cno-identity s = Σ (s' : ProgramState) , (id s' ≡ s)
```

This simplifies to a singleton fiber containing only the original state `s`.

## Core Theorems

### Theorem 1: CNO-Singleton Equivalence

```agda
cno-echo-equivalence : ∀ (s : ProgramState) → 
                        Echo cno-identity s ≃ ((s' : ProgramState) × (s' ≡ s))
```

**Proof**: The echo over identity is equivalent to a singleton pair `(s, refl)`.

### Theorem 2: All CNOs are Echoes

```agda
all-cnos-are-echos : ∀ (p : ProgramState → ProgramState) → 
                      (∀ σ → p σ ≡ σ) → 
                      ∀ σ → Echo p σ ≃ Echo id σ
```

**Proof**: Any state-preserving program has echoes isomorphic to identity echoes.

### Theorem 3: CNO Composition

```agda
cno-composition-echo : ∀ (s : ProgramState) (e : Echo cno-identity s) → 
                        map-over (id , (λ x → refl)) e ≡ e
```

**Proof**: Composing identity functions preserves echo structure.

### Theorem 4: Bennett-reversible erasure cost (FINITE DOMAIN ONLY)

```agda
-- in EchoThermodynamics
bennett-reversible :
  (f : Fin n → B) (y : B) (_≟_ : DecidableEquality B) (T : Temperature) →
  FiberSize-fin f y _≟_ ≡ 1 →
  fiber-erasure-bound f y _≟_ T ≡ 0
```

**Status: PROVED**, and (2026-05-18) generalised three times: from
the identity to *every injective map* (`bennett-reversible-injective`),
from `Fin n` to *any Bishop-finite carrier* via transport
(`bennett-reversible-finite`, `EchoThermodynamicsFinite`), and from
there to an *arbitrary carrier with no finiteness at all*
(`bennett-reversible-arbitrary`, `EchoThermodynamicsArbitrary`),
including the genuine infinite `Program` CNO
(`bennett-reversible-cno-identity`). `bennett-reversible-id-zero` is a
corollary. The Bennett zero direction is closed for *all* carriers.

**The infinite-carrier _quantitative collapse_ functional is
`[CLOSED-NEG]`**: not a vague gap and no longer open — the falsifiable
obligation **O-THERMO-∞** is now discharged *negatively*, its
impossibility mechanised (`collapse-cost-impossible`,
`EchoThermoCollapseImpossible`; see §"Thermodynamic Bridge" below). A
count-based collapse cost provably cannot exist on an infinite carrier
as a total `--safe` function — a settled boundary, do not reopen. The
earlier `cno-thermodynamic-optimality` formulation rested on a
hardcoded `FiberSize ≡ 1` that made the claim vacuous; that hardcode
is gone, the honest finite/Bishop-finite/arbitrary cases are proved,

## Mapping to Absolute Zero's CNO Definition

Absolute Zero defines CNOs with four properties:

1. **Termination**: `Terminates(p, σ)`
2. **State Preservation**: `FinalState(p, σ) = σ`
3. **No Side Effects**: `NoSideEffects(p)`
4. **Thermodynamic Reversibility**: `ThermodynamicallyReversible(p)`

### Echo Type Interpretation

| Absolute Zero Property | Echo Type Interpretation |
|------------------------|--------------------------|
| `Terminates(p, σ)` | `Echo p σ` is inhabited |
| `FinalState(p, σ) = σ` | `Echo p σ` contains `(σ, refl)` |
| `NoSideEffects(p)` | `Echo p σ` is singleton (only `σ` maps to `σ`) |
| `ThermodynamicallyReversible(p)` | `Echo p σ ≃ Echo id σ` |
| `ThermodynamicOptimality(p)` | `reversible-erasure-cost p σ d T ≡ 0` _(proved for every injective `p` on **any** carrier — Bishop-finite via `fiber-erasure-bound`, arbitrary/infinite incl. the `Program` CNO via `reversible-erasure-cost` — Theorem 4; the quantitative-collapse functional on an infinite carrier is settled `[CLOSED-NEG]` — impossibility mechanised, `collapse-cost-impossible`)_ |
| `InformationPreservation(p)` | _no formalised analogue — `echo-information-loss` was never defined; see Theorem 5_ |
| `UniversalCNO(p)` | `∀ σ → Echo p σ ≃ Echo id σ` (NEW) |
| `ModelIndependence(p)` | `Echo f y ≃ Echo f' y'` across isomorphic models (NEW) |

## Categorical Bridge (NEW)

The `EchoCategory.agda` module establishes a comprehensive categorical formalization of echo types, directly supporting Absolute Zero's goal of universal CNO definitions through category theory.

### Key Concepts

1. **Echo Fibration**: Echo types form a fibration where each morphism `f : A → B` has fibers `Echo f y` over each `y : B`
2. **Echo Category**: Category where objects are echo types and morphisms preserve echo structure
3. **Universal CNO**: CNOs as identity morphisms that preserve echo structure: `∀ s → Echo p s ≃ Echo id s`
4. **Model Independence**: Echo types are preserved across isomorphic computational models

> **Scope note (2026-05-18).** "Model Independence" *here* means the
> bare-`Echo` invariance `Echo f y ≃ Echo f' (proj₁ (isoB y))` under an
> isomorphism of the underlying objects (the `ModelIndependence`
> theorem below). This is a distinct, valid statement and is **not**
> affected by retraction R-2026-05-18, which retracted the *graded
> modality's* "two independent models / model-independence" claim
> (`GradedLossModel`/`EchoRelModel`). Do not conflate the two; this
> CNO consumer-bridge doc is firewalled from the identity claim. See
> `docs/retractions.adoc`.

### Core Theorems

```agda
-- Echo types form a fibration
EchoFibration : Fibration
EchoFibration {A} {B} f y = Echo f y

-- Universal CNO definition (matches Absolute Zero's goals)
UniversalCNO : ∀ {A : Ob} (p : A → A) → Set
UniversalCNO p = ∀ (s : A) → Echo p s ≃ Echo id s

-- Categorical equivalence with Absolute Zero's definition
CNO-Category-Equivalence : ∀ {A : Ob} (p : A → A) → 
                           UniversalCNO p ≃ AbsoluteZeroUniversalCNO p

-- Model independence across isomorphic models
ModelIndependence : ∀ {A B A' B' : Ob} (f : A → B) (f' : A' → B') 
                    (isoA : A ≃ A') (isoB : B ≃ B') 
                    (commute : ∀ a → f' (proj₁ (isoA a)) ≡ proj₁ (isoB (f a))) 
                    {y : B} → 
                    Echo f y ≃ Echo f' (proj₁ (isoB y))
```

### Connection to Absolute Zero Goals

1. **Universal CNO Definition**: Provides categorical foundation for `AbsoluteZeroUniversalCNO`
2. **Model Independence**: Supports cross-language CNO verification
3. **Category Theory Module**: Formalizes echo types as fibrations
4. **Functorial Properties**: Proves echo maps preserve categorical structure

## Thermodynamic Bridge — finite-domain Landauer/Bennett bounds

The `EchoFiberCount.agda` + `EchoThermodynamics.agda` pair
formalises the *shape* of Landauer's bound at the finite-domain
`Fin n → B` level. The earlier draft of this section claimed
significantly more than was proved (a `FiberSize ≡ 1` hardcode
made `cno-zero-energy` and `cno-zero-information-loss` vacuous);
those claims have been retracted and replaced with what is
actually in the proven suite.

### Key Concepts

1. **Honest fiber count**:
   `FiberSize-fin f y _≟_` enumerates `Fin n` and asks the
   decidable equality at each index — actual preimage count, not
   a hardcoded constant.
2. **Landauer bound shape**:
   `landauer-bound T n = k * T * ⌊log₂ n⌋`. Linear in T,
   logarithmic in the alternatives count. `⌊log₂ 0⌋ = ⌊log₂ 1⌋ = 0`
   discharges the trivial corners.
3. **Fiber erasure bound**:
   `fiber-erasure-bound f y _≟_ T = landauer-bound T (FiberSize-fin f y _≟_)`.

### Core Theorems (proved)

```agda
-- Bennett: if the fiber size at y is 1, the erasure bound is 0
bennett-reversible :
  (f : Fin n → B) (y : B) (_≟_ : DecidableEquality B) (T : Temperature) →
  FiberSize-fin f y _≟_ ≡ 1 →
  fiber-erasure-bound f y _≟_ T ≡ 0

-- Identity at index zero — canonical instance
bennett-reversible-id-zero :
  (_≟_ : DecidableEquality (Fin (suc m))) (T : Temperature) →
  fiber-erasure-bound (λ x → x) zero _≟_ T ≡ 0

-- Landauer worst case: every input maps to the same y
landauer-collapse :
  (f : Fin n → B) (y : B) (_≟_ : DecidableEquality B) (T : Temperature) →
  (∀ x → f x ≡ y) →
  fiber-erasure-bound f y _≟_ T ≡ k * T * ⌊log₂ n⌋
```

### The C+ gap, and how far the bound generalises (2026-05-18)

The `STABILITY_ANALYSIS.md` §3.3 ~70% / C+ rating was driven by a
single concrete gap: the only proved zero-cost (Bennett) instance was
`bennett-reversible-id-zero` — *the identity map, at index zero*. The
general predicate `ThermodynamicOptimality(p) := fiber-erasure-bound
p σ _≟_ T ≡ 0` (Absolute-Zero mapping table) was otherwise unproved
off that one point. Two strictly stronger cases are now closed, and
the residual is pinned to one falsifiable obligation.

**Proved — generalisation past the identity (`EchoFiberCount`,
`EchoThermodynamics`).** Zero erasure cost holds for *every injective
map*, at *every value it hits*, not just `id`/`zero`:

```agda
FiberSize-fin-injective :
  (f : Fin n → B) (y : B) (_≟_ : DecidableEquality B)
  (inj : ∀ {i j} → f i ≡ f j → i ≡ j)
  (i₀ : Fin n) → f i₀ ≡ y →
  FiberSize-fin f y _≟_ ≡ 1

bennett-reversible-injective :
  (f : Fin n → B) (y : B) (_≟_ : DecidableEquality B)
  (inj : ∀ {i j} → f i ≡ f j → i ≡ j)
  (i₀ : Fin n) → f i₀ ≡ y → (T : Temperature) →
  fiber-erasure-bound f y _≟_ T ≡ 0
```

This is the exact constructive content of "reversible ⇒ no
Landauer-mandated dissipation": injectivity *is* reversibility, a
singleton fiber *is* the absence of fan-in. `bennett-reversible-id-zero`
is now a corollary, not the only instance.

**Proved — generalisation past `Fin n` to any Bishop-finite carrier
(`EchoThermodynamicsFinite`).** What blocks `Fin n → B` from
generalising is *exactly* one thing: `fiber-erasure-bound` is routed
through `FiberSize-fin`, whose totality is structural recursion on the
index bound `n` of `Fin n`. That is removed for any carrier `A`
carrying an explicit bijection `A ≃ Fin n` (record `FiniteDomain`),
by transport along the bijection — no new axiom, `--safe --without-K`,
zero postulates:

```agda
record FiniteDomain (A : Set) : Set where
  field card : ℕ; to : A → Fin card; from : Fin card → A
        from∘to : ∀ a → from (to a) ≡ a
        to∘from : ∀ i → to (from i) ≡ i

bennett-reversible-finite :
  (fd : FiniteDomain A) (f : A → B) (y : B) (_≟_ : DecidableEquality B)
  (inj : ∀ {a a′} → f a ≡ f a′ → a ≡ a′)
  (a₀ : A) → f a₀ ≡ y → (T : Temperature) →
  fiber-erasure-bound-fin fd f y _≟_ T ≡ 0

landauer-collapse-finite :
  (fd : FiniteDomain A) (f : A → B) (y : B) (_≟_ : DecidableEquality B) →
  (∀ a → f a ≡ y) → (T : Temperature) →
  fiber-erasure-bound-fin fd f y _≟_ T ≡ k * T * ⌊log₂ card fd ⌋
```

**Proved — Bennett zero-cost on an _arbitrary_ carrier, past
Bishop-finite (`EchoThermodynamicsArbitrary`).** The finiteness in the
two results above is an artefact of routing cost through
`FiberSize-fin` (an enumeration); the *physical content* of Bennett —
a reversible map has no fan-in — needs no finiteness at all. An
injective `f : A → B` has a subsingleton fiber for *any* `A : Set a`.
Keying the Bennett cost on a decidable-inhabitation occupancy instead
of an enumeration discharges this with no finiteness hypothesis,
`--safe --without-K`, zero postulates:

```agda
FiberSubsingleton f y = (p q : Echo f y) → proj₁ p ≡ proj₁ q  -- K-free

injective⇒fiber-subsingleton :                                -- any A
  (inj : ∀ {x x′} → f x ≡ f x′ → x ≡ x′) → FiberSubsingleton f y

bennett-reversible-arbitrary :        -- A : Set a arbitrary; no Fin, no ≃
  (inj : ∀ {x x′ : A} → f x ≡ f x′ → x ≡ x′)
  (d : Dec (Echo f y)) (T : Temperature) →
  reversible-erasure-cost f y d T ≡ 0
```

This is **not** a vacuous re-definition: `occupancy≡FiberSize-fin`
proves the occupancy *coincides with* the established honest count
`FiberSize-fin` on every finite restriction under the subsingleton
hypothesis, and `bennett-arbitrary-refines-finite` shows it strictly
subsumes the `Fin n` result of PR&nbsp;#58. The `--without-K` reason
for the first-projection form (not full fiber `isProp`) is recorded in
the module header: full `isProp` needs `B` an h-set / UIP, unavailable
under `--without-K`; the projection form is K-free provable from
injectivity and is exactly what the count needs.

**The real CNO theorem, on the infinite carrier.** `cno-identity :
Program → Program` (the absolute-zero `Program = List Instruction`
carrier — *infinite*, the very `ProgramState`-flavoured type the
original companion remarks disclaimed) is injective, so
`bennett-reversible-cno-identity` gives erasure cost `≡ 0` at every
fiber and temperature. "A CNO dissipates zero Landauer energy" is now
a *real* theorem for the identity/injective CNO over the genuine
infinite carrier — no longer the vacuous claim the deleted
`ECHO-CNO-BRIDGE-SUMMARY.md` made.

## Practical Implications

### 1. CNO Verification via Echo Types

Instead of proving four separate properties, we can verify CNOs by:
1. Showing the program's echo is inhabited
2. Proving the echo is isomorphic to identity echoes
3. Verifying singleton property

### 2. Echo Type Theory as Foundation

The bridge shows that echo type theory provides a unifying framework:
- CNOs become a special case of echo types
- Echo composition laws apply to CNO composition
- Echo preservation theorems apply to CNO state preservation

### 3. Cross-Repository Integration

The bridge enables:
- Using echo-type proofs in CNO verification
- Applying CNO analysis to echo-type systems
- Shared theorem libraries between repositories
