== Where echo-types sits in this programme

`echo-types` is the **verified proof companion**. It does not host the Oikos
compiler (that is Rust, in the `oikos` repo). echo-types supplies the
machine-checked *principle* the MUST's economics reading rests on: that
micro→macro aggregation is a structured-loss (`Echo`) map with **no canonical
disaggregation** — there is no left inverse (section) recovering the micro
ledger from the macro total. That is the Sonnenschein–Mantel–Debreu /
representative-agent critique stated type-theoretically, and as of 2026-06-18
(issue #175) it is the *general* monoid/group aggregation theorem, with the
macro economics as one instance (`Example-PairSum`).

== Honest cross-cutting notes

* The normative MUST is **real and tested at the check layer on the AST** (the
  Godley/instrument/currency/cross-ref tests build hand-constructed models and
  assert reject-on-violation). It is *not yet* demonstrated end-to-end from
  surface source through the parser and desugar, because desugar is blocked.
* echo-types contributes the *principle*, not the *enforcement*: there is no
  Agda↔Rust import path. The bridge is citation-level by design.
