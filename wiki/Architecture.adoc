= Architecture
:toc: macro
:toclevels: 3

[.lead]
echo-types is organised as three proof workstreams over a shared core, plus a
cross-repo bridge surface. All of it lives under `proofs/agda/` and builds as a
single verified suite.

toc::[]

== The shared core

[cols="1,3", options="header"]
|===
| Module | Role

| `Echo.agda`
| The core definition `Echo f y := Σ (x : A) , (f x ≡ y)`, plus `echo-intro`.
  The *stdlib-only root* of the kernel cone (imports zero in-repo modules).

| `EchoKernel.agda`
| The second (and only other) member of the funext-free kernel cone; its
  in-repo imports are a subset of `{ Echo }`. Enforced by `scripts/kernel-guard.sh`.

| `All.agda`
| The import-everything target. Every module must be wired in here; orphans
  that compile but are absent are treated as dead code.

| `Smoke.agda`
| Pins every headline theorem via a `using` clause, each in its own
  open-import block with a header comment.

| `Echo.*` (under `proofs/agda/Echo/`)
| The *foundation contract surface* — the curated, stable interface
  downstream languages build against: `Echo.Index.ThinPoset` (echo index),
  `Echo.Modality.{Interface,Core}` (the modality, measure-independent),
  `Echo.Separation.NotResourceInstance` (the *Echo IS-NOT a resource
  instance* invariant), `Echo.Measure.{Interface,Examples}` (the
  residue-measure observation seam). Contract: `FOUNDATION_CONTRACT.md`.
|===

== Track 1 — Composition (landed)

* Cancellation iso `cancel-iso`, parameterised by `s-left` / `s-right` and both
  triangle identities (PR #25).
* Pentagon coherence: `Echo-comp-iso-pent-{B,echo}` as `refl`; the full
  Σ-associativity iso `Echo-comp-pent-Σ-assoc` packaged via stdlib
  `Function.Bundles._↔_`.
