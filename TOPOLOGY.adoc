== Cross-repo dependencies

== Cross-repo consumers

This is a foundation library; consumers are downstream Agda formalisations and design docs that cite the echo construction. Currently none formally wire `echo-types` as a dependency, though `coord-mcp` has a TODO for dogfooding it (per `hyperpolymath/standards/docs/archive/provenance/coord-mcp/COORD-MCP-TODO.md`).

== Verification posture

|===
| Track | Soundness | Notes

| Core echo theorems | constructive Agda, no postulates | `echo-intro`, `map-over`, `map-over-id`, `map-over-comp`, `map-square`
| Characteristic family | constructive | `collapse-non-injective`, `no-section-*`, `echo-true≢echo-false`, `visible-constraint`
| Bridge modules | constructive (per bridge) | linear / graded / tropical / choreo / epistemic / CNO / Janus / Dyadic / Ordinal
| Ordinal Buchholz track | mostly admitted, some closed | `WellFounded`, extended-order tower; closure pending
| EI-2 (integration recipe) | TERMINATED NEGATIVELY (PATH B) | Per-axis halves proved; full 2D iff theorem parked as not formalisable in safe Agda without postulates
|===
